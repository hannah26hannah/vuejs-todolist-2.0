# vue-todolist-2

[Demo](https://hannah26hannah.github.io/vuejs-todolist-2.0/)

## Used
- `Vue.js`
- `TypeScript`
- `TailWinds`
- Handling SVG Graphics & Animation with `anime.js`
- Manage state with `Vuex`
- Display data with `chart.js` reactively
